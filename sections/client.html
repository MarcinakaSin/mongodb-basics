MongoDB Client

Using the MongoDB Client

> use foobar
switched to db foobar

> db
foobar

> db.blog.remove({title: "My Blog Post"})
deletes from collection

> db.blog.insert(data)
stores data in the blog collection

> db.blog.find()
returns all data in the blog collection

> help
provides helpful content :)

> post = { "title": "My Blog Post",
... "content": "Here's my blog post.",
... "date": new Date() }
> post.comments = []
> db.blog.update({title: "My Blog Post"}, post)
update takes two parameters:
1. Criteria to find which document to update.
2. The new document.
